# Avatar assets

Final avatar artwork should live here as transparent PNG files.

Recommended format:

- 512x512 PNG
- transparent background
- centered character at around 80% of the canvas
- readable at 44px, 64px, and 96px
- illustrated "monito" style with soft marker/pencil edges, solid fills, and subtle diagonal shading

Current catalog status:

- `devil.png` and `joker.png` are the active illustrated avatars.
- Add new files only when they match the style above, then add their entries to `src/data/avatarCatalog.ts` and `src/i18n/resources.ts`.
